What is the difference between innerHTML, innerText, and textContent?

The difference between innerHTML, innerText, and textContent is how they handle content. innerHTML changes or gets HTML content but can be unsafe. innerText gets only visible text and may slow down the page. textContent gets all text, even hidden text, and is the fastest option.

What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to each child, we add one listener to the parent and check which child was clicked. This makes the code simpler, faster, and useful for dynamic elements

What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts from the target element and then bubbles up through its parent elements. When an event occurs, it first triggers on the element that was clicked or interacted with, and then it moves up to the parent elements, calling any event listeners attached to them.

How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement(). To add it to the page, use appendChild() or insertBefore() on a parent element. To remove it, use removeChild() on the parent.
